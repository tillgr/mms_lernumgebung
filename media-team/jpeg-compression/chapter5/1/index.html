5.1 Entropiekodierung: Übersicht

Nachdem die Informationen in der vorhergehenden Phase auf das Nötigste beschränkt worden sind, sollen sie in dieser abschließenden Phase der Kodierung noch möglichst effizient gespeichert werden. Effizient bedeutet in diesem Fall, dass alle mehrfach auftretenden, das heißt redundanten Informationen zusammengefasst werden. Man bezeichnet dieses Vorgehen daher als Redundanzreduktion. Die Phase wird Entropiekodierung genannt. Entropie ist ein Maß für den Informationsgehalt, der bei redundanten Informationen gleich null ist und somit keinen weiteren Wissens-zuwachs bedeutet.
