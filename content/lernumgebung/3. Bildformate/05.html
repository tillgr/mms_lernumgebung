(Bearbeitungs-) Ebenen

Bildeditoren, wie Photoshop, unterstützen die Darstellung von Bildteilen oder Bildern in verschiedenen, sich überlagernden Ebenen.

{{% image url="Bildformate/5/2_4_1_ebenen.jpg" alt="Wirkung von Ebenen" caption="Abb. 3.5-0 Wirkung von Ebenen" %}}

Ebenen ermöglichen es, spezielle Bildbereiche zu bearbeiten, ohne dass andere Informationen verändert werden. Eine Ebene lässt sich z.B. unabbhängig von anderen verschieben. Alle Ebenen eines Bildes haben die gleiche Anzahl Pixel, Kanäle sowie denselben Bildmodus, z.B. RGB oder CMYK.

Ebenen können zwischen Dokumenten ausgetauscht werden, z.B. mit Hilfe der Ebenen-Palette. Sie können in ihrer Lage verändert oder vorübergehend ausgeblendet werden.

{{% image url="Bildformate/5/2_4_2_moduswirkungen_auf_ebenen.jpg" alt="Moduswirkungen auf Ebenen" caption="Abb. 3.5-1 Moduswirkungen auf Ebenen" %}}

Attribute einer Ebene sind ihr Deckungsgrad und der Ebenen-Modus. Jeder Ebene kann eine bestimmte Deckungskraft zugeordnet werden. Ebenen-Modi legen fest, wie die Pixel in der Ebene im Verhältnis zu den dahinter liegenden Pixeln anderer Ebenen erscheinen. Eine Ebene besteht aus transparenten, opaken (nicht transparenten) und halbtransparenten Bereichen, aus deren Deckkraft eine Transparenzmaske berechnet wird. Zusätzlich kann eine Ebene auch um eine Ebenenmaske ergänzt werden, welche die maskierten Teile der Ebene ausblendet, so dass sie im Gesamtbild nicht mehr sichtbar sind. Sie wirkt wie ein Passepartout aus schwarzer Pappe, das Löcher besitzt, durch die ein darunterliegendes Bild hindurchscheint. Eine Ebenenmaske ist nur in der Ebene wirksam, der sie zugewiesen ist. Andere Ebenen werden von ihr nicht verdeckt.

Kanäle

Jedes Bild kann ein oder mehrere Kanäle enthalten. Ein CMYK-Bild hat z.B. mindestens vier Kanäle, jeweils einen für Cyan, Magenta, Gelb und Schwarz.

Außerdem besitzt ein Bilddokument einen Gesamtkanal, der alle Farbinformationen anzeigt. Zudem können weitere Kanäle zum Aufbewahren und Bearbeiten von Masken definiert werden, sog. Alpha-Kanäle. Durch Kanalberechnungen können Pixelwerte aus zwei Kanälen überlagert und mit unterschiedlichen Methoden kombiniert werden.

Alpha-Kanal: Ein Erklärungsmodell von Alpha-Kanälen kann sein

  • Ausgangspunkt: Graustufenfoto auf einer Folie und zusätzliche nicht bedruckte Folien, die jeweils einem Kanal entsprechen.
  • Schneidet man in eine Folie ein Loch und legt diese auf die Graustufenfolie, kann im Bereich des Loches das Bild bearbeitet werden, der Rest des Bildes bleibt geschützt. Mit verschiedenen Folien unterschiedlicher Löcher wird der Arbeitsbereich eingeschränkt.
  • Das Graustufenbild entspricht dem Schwarz-Kanal, durchlöcherte Folien einem Alpha-Kanal, der einen gespeicherten Auswahlbereich, eine Maske enthält.

Enthält ein Bild einen Alpha-Kanal, so wird zu jedem einzelnen Bildpunkt zusätzlich ein Alpha-Wert gespeichert. Der Alpha-Kanal kann als Transparenz-Maske angesehen werden. Im Folgenden wird die Anwendung eines Alpha-Kanals demonstriert. Bild (b) enthält den in (c) abgebildeten Alpha-Kanal und wird gegen den Hintergrund (a) dargestellt. Das Ergebnis ist in (d) zu sehen.

{{% image url="Bildformate/5/2_4_4_alpha-kanal_1.jpg" alt="Anwendung eines Alphakanals" caption="Abb. 3.5-2 Anwendung eines Alphakanals" %}}

Kanäle werden z.B. im TIFF- und PNG-Dateiformat mit den Bilddaten gespeichert. Im JPEG-Format wird kein Alpha-Kanal gespeichert.

Kanäle werden bei Photoshop in der Kanal-Palette verwaltet, können ein- und ausgeblendet, dupliziert und gemischt werden. Es ist möglich, Farben, Maskierungen und Deckkraft für jeden Kanal festzulegen.

Jedes Bild kann bis zu 24 Kanäle enthalten.
