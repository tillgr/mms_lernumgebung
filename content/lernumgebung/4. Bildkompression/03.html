4.3.1 Differenzkodierung

Unterscheiden sich die Werte einer Folge nur geringfügig voneinander, z.B. die Farbwerte einer homogenen Hintergrundfläche, so bietet sich die Kodierung der Differenz an.

Bei Bewegtbildsequenzen wird in der zeitlichen Folge die Differenz zwischen benachbarten Frames kodiert.

4.3.2 Delta Modulation

Bei diesem Verfahren handelt es sich um eine Abwandlung der DPCM, die sich dadurch unterscheidet, dass die benachbarten Abtastwerte nur um die Werte +1 oder -1 von dem realen Abtastwert abweichen können. Für die Kodierung benachbarter Abtastwerte wird lediglich ein Bit verwendet. Es repräsentiert p Stufen.

Durch eine Erhöhung der Abtastrate können schnelle und hohe Signaländerungen erfasst werden.

Beispiele: p = 3

  • unkomprimiert: 112, 114, 117, 115, 111, 109, ....
  • komprimiert: 1, 1, 0, 0, 0, ...
  • dekomprimiert: 112, 115, 118, 115, 112, 109, ...

4.3.3 Adaptive Kompression

Bei adaptiver Kompression wird das Verfahren an die zu komprimierenden Daten angepasst.

Adaptive DPCM: ADPCM

Der Kodierer dividiert den DPCM-Wert durch eine geeignete Konstante und der Dekoder multipliziert die komprimierten Werte wieder mit dieser Konstante. Für Signale mit großen DPCM-Werten (großer Anteil hoher Frequenzen) wird eine Konstante mit großem Wert gewählt. Damit ergibt sich eine grobe Quantisierung, d.h. niederfrequente Anteile bleiben unberücksichtigt. Bei Signalen mit kleinen DPCM-Werten wird eine kleine Konstante gewählt, d.h. es ergibt sich eine gute Auflösung niederfrequenter Signalanteile. Treten in diesem Fall plötzlich große Werte auf, so entstehen Signalverzerrungen (Slope Overload).
